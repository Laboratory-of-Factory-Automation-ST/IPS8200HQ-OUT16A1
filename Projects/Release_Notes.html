Purpose

The projects are a set of applicative tasks that aim to explain and provide use cases of the different products features based on the products hardware (Boards, STM32 Core(s) features, interconnections and peripherals) and built around the different firmware components. They can be directly used or adapted by the STM32CubeF4 or STM32CubeG4 Firmware Package users. They provide basic functionalities, show main features or demonstrate performance. They are provided with preconfigured projects for the main supported toolchains.

In the STM32CubeF4 or STM32CubeG4 Firmware Package, the projects are organized in a sequential scheme based on the following levels:

  • Boards: Hardware platform (evaluation, discovery, nucleo boards)
  • Project Category:
    • Examples

This software provides a set of examples useful to experiment how to correctly manage the Intelligent Power Switch family of STMicroelectronics products.

Currently available code can be used with the following products:

  • ISO8200BQ, a Galvanic isolated Octal High-Side smart power solid state-relay (Smart driving 0.7A per channel)
  • IPS2050H, a Dual High-Side Switch (Smart driving 2.5A loads)
  • IPS2050H-32, a Dual High-Side Switch (Smart driving 5.7A loads)
  • IPS1025H, a Single High-Side Switch (Smart driving 2.5A loads)
  • IPS1025H-32, a Single High-Side Switch (Smart driving 5.7A loads)
  • IPS4260L, a Quad Low-Side Switch (Smart driving 0.5A loads)
  • IPS160HF, a Single High-Side Switch (Smart driving 2.5A loads)
  • IPS8160HQ, an Octal High-Side Switch (Smart driving 0.7A loads)
  • IPS161HF, a Single High-Side Switch (Smart driving 0.7A loads)
  • ISO808, a Galvanic Isolated Octal High-Side Power Solid State Relay for high inductive loads (Smart driving 0.7A loads)
  • ISO808A, a Galvanic Isolated Octal High-Side Power Solid State Relay with SPI interface (Smart driving 0.7A loads)
  • ISO808-1, a Galvanic Isolated Octal High-Side Power Solid State Relay for high inductive loads (Smart driving 1.0A loads)
  • ISO808A-1, a Galvanic Isolated Octal High-Side Power Solid State Relay with SPI interface (Smart driving 1.0A loads)
  • IPS1025HF, a Single High-Side Switch (Smart driving 2.5A loads)
  • IPS8200HQ, an Octal High-Side smart power solid-state relay with serial/parallel selectable interface on-chip (Smart driving 0.7A loads)
  • IPS8200HQ-1, an Octal High-Side smart power solid-state relay with serial/parallel selectable interface on-chip (Smart driving 1.0A loads)
  • IPS8160HQ-1, an Octal High-Side Switch (Smart driving 1.0A loads)
