Purpose

The X-CUBE-IPS software package is an expansion for STM32Cube. This software provides drivers running on STM32 for STMicroelectronics:

  • ISO8200BQ, a Galvanic isolated Octal High-Side smart power solid state-relay (Smart driving 0.7A per channel)
  • IPS2050H, a Dual High-Side Switch (Smart driving 2.5A loads)
  • IPS2050H-32, a Dual High-Side Switch (Smart driving 5.7A loads)
  • IPS1025H, a Single High-Side Switch (Smart driving 2.5A loads)
  • IPS1025H-32, a Single High-Side Switch (Smart driving 5.7A loads)
  • IPS4260L, a Quad Low-Side Switch (Smart driving 0.5A per channel)
  • IPS160HF, a Single High-Side Switch (Smart driving 2.5A loads)
  • IPS8160HQ, an Octal Channel High-Side driver (Smart driving 0.7A per channel)
  • IPS161HF, a Single High-Side Switch (Smart driving 0.7A loads)
  • ISO808, a Galvanic Isolated Octal High-Side Power Solid State Relay for high inductive loads (Smart driving 0.7A per channel)
  • ISO808A, a Galvanic Isolated Octal High-Side Power Solid State Relay with SPI interface for high inductive loads (Smart driving 0.7A loads)
  • ISO808-1, a Galvanic Isolated Octal High-Side Power Solid State Relay for high inductive loads (Smart driving 1.0A per channel)
  • ISO808A-1, a Galvanic Isolated Octal High-Side Power Solid State Relay with SPI interface for high inductive loads (Smart driving 1.0A loads)
  • IPS1025HF, a Single High-Side Switch (Smart driving 2.5A loads)
  • IPS8200HQ, an Octal High-Side smart power solid-state relay with serial/parallel selectable interface on-chip (Smart driving 0.7A loads)
  • IPS8200HQ-1, an Octal High-Side smart power solid-state relay with serial/parallel selectable interface on-chip (Smart driving 1.0A loads)
  • IPS8160HQ-1, an Octal Channel High-Side driver (Smart driving 1.0A per channel)

The X-CUBE-IPS is built on top of STM32Cube software technology that eases portability across different STM32 microcontrollers.

X-CUBE-IPS software features:

  • Driver layer for complete management of ISO8200BQ, IPS2050H, IPS2050H-32, IPS1025H, IPS1025H-32, IPS4260L, IPS160HF, IPS8160HQ, IPS161HF, ISO808, ISO808A, ISO808-1, ISO808A-1, IPS1025HF, IPS8200HQ, IPS8200HQ-1 and IPS8160HQ-1 devices
